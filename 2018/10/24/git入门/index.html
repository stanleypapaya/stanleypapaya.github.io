常见git命令的使用

  • git init

    • 将当前目录即demo-1初始化为一个本地仓库
    • 该仓库主要用于远端服务器管理代码
    • 保存git历史提交的版本信息，防止冲突

  • git add .

    • 将当前目录所有变动的文件加到暂存区
    • 查看状态后发现文件名左边有绿色A
    • 绿色A表示文件已经被加到暂存区

  • git commit -v

    • 将暂存区的文件提交到.git仓库
    • 表示三个文件（index.html, main.js, style.css)被提交
    • 提交的信息是‘first commit’
