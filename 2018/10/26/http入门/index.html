WWW的历史

1990年

万维网（world wide web）诞生
Tim Berners-Lee 发明了用网址就能访问页面的方法，他发明了第一个网页，第一个浏览器和服务器

1990年之前

  • 1965年， Email被发明出来
  • 1971年， Email开始使用@符号
  • 1979年， 邮件讨论组被发明出来，人们在一个话题下可以公开互发邮件
  • 1980年， HTTP被提出

HTTP请求的组成

  1. 动词 路径 协议/版本
  2. Key1: value1
  3. Key2: value2
  4. Key3: value3
  5. Content-Type: application/x-www-form-urlencoded
  6. Host: www.baidu.com
  7. User-Agent: curl/7.54.0
  9. 要上传的数据

请求最多包含4个部分（POST），最少包含3个部分（如GET），第三部分永远是一个回车（\n)

HTTP响应的组成

  1. 协议/版本号 状态码 状态解释
  2. Key1: value1
  3. Key2: value2
  4. Content-Length: 17931
  5. Content-Type: text/html
  7. 要下载的内容

上面例子中的状态码 302 moved temporarily 表示资源的URI临时换位置

用Chrome开发者工具查看HTTP请求内容

1. 鼠标右键，点击Inspect，打开Network

2. 地址栏输入网址，在Network点击查看Request

3. 点击【view resource】

4. 可以看到请求的3个部分，如果有第4部分，则可以再FormData里看到

用Chrome开发者工具查看HTTP响应内容

1. 打开Network

2. 地址栏输入网址，在Network点击第一个响应

3. 查看Respond Headers，点击【view resource】

4. 可以看到响应的前2个部分

5. 查看Response，你会看见响应的第4部分
