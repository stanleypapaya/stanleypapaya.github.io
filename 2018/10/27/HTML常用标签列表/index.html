HTML常用标签列表

tag name description
<ul> 代表无序列表
<a> 代表一个链接到其他资源的超链接
<strong> 代表特别重要的文字
<p> 代表段落
<nav> 代表导航条
<link> 代表链接外部的CSS到文档
<style> 用于内联CSS
<header> 用于页面或章节的头部
<h1> 用于最大的标题
<footer> 用于页面或章节的尾部
<main> 用于文档主要或重要的内容
<img> 用于一张图片
<video> 用于一段视频，并提供播放界面
<canvas> 代表位图区域，可以通过脚本在它上面实时呈现图形
<svg> 定义一个嵌入式矢量图
