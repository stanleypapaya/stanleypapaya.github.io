JavaScript的数据类型

Posted on 2018-11-06

JS数据类型分类

JavaScript中的数据类型分为数字（Number），字符串（String），布尔型（Bool），Null，Undefined和对象（Object）

Number

JavaScript 内部，所有数字都是以64位浮点数形式储存,整数也是如此。

数值的进制

  • 十进制：没有前导0的数值。
  • 八进制：有前缀0o或0O的数值，或者有前导0、且只用到0-7的八个阿拉伯数字的数值。
  • 十六进制：有前缀0x或0X的数值。
  • 二进制：有前缀0b或0B的数值。

默认情况下，JavaScript 内部会自动将八进制、十六进制、二进制转为十进制

NaN

NaN是 JavaScript 的特殊值，它属于Number类型，但表示“非数字”（Not a Number）

  • 主要出现在将字符串解析成数字出错的场合
  • 另外，一些数学函数的运算结果会出现NaN

String

字符串就是零个或多个排在一起的字符，放在单引号或双引号之中。

转义符

（\）在字符串内有特殊含义，用来表示一些特殊字符，所以又称为转义符,在字符串中不占length

  • \0 ：null（\u0000）
  • \b ：后退键（\u0008）
  • \f ：换页符（\u000C）
  • \n ：换行符（\u000A）
  • \r ：回车键（\u000D）
  • \t ：制表符（\u0009）
  • \v ：垂直制表符（\u000B）
  • \’ ：单引号（\u0027）
  • \” ：双引号（\u0022）

字符串多行书写

在表达字符串时常常因为长度或者易读性问题需要换行，一般连接运算符（+）可以连接多个单行字符串，将长字符串拆成多行书写。输出时为单行

Bool

布尔值代表“真”和“假”两个状态。“真”用关键字true表示，“假”用关键字false表示。布尔值只有这两个值。

6个falsy值

在javaScript中只有以下6个值为false，其他都为true

  • 0
  • ‘’和””
  • false
  • null
  • undefined
  • NaN

Undefined 和 Null

null与undefined都可以表示“没有”，含义非常相似，但在用法上有细微的区别

  • 如果一个变量(非object）没有赋值 -undefined

  • 如果有一个对象（object）没有赋值 -null
