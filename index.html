请说出至少三种排序的思路，这三种排序的时间复杂度分别为

1.O(n*n)
2.O(n log2 n)
3.O(n +max)

1.冒泡排序

  • 从第一个元素开始，比较相邻的一对元素。
  • 如果前面的比后面的大，交换它们。
  • 一轮遍历之后最大的元素一定会位于序列结尾。
  • 重复1-3，每次将剩下的元素中最大的放到剩下元素组成序列的结尾。

2.快速排序

  • 选取一个基准元素（pivot）
  • 比pivot小的放到pivot左边，比pivot大的放到pivot右边
  • 对pivot左边的序列和右边的序列分别递归的执行步骤1和步骤2

3.计数排序

  • 根据最大和最小值构建从小到大的计数槽
  • 遍历原数组，将元素扔进对应的计数槽
  • 遍历全部数字即完成计数，取出就是从小到大的顺序

一个页面从输入 URL 到页面加载显示完成，这个过程中都发生了什么？

1.根据域名查找IP地址

  • 浏览器搜索自己的 DNS 缓存；
  • 搜索操作系统中的 DNS 缓存；
  • 搜索操作系统的 hosts 文件；
  • 操作系统将域名发送至 LDNS（如电信）
  • LDNS 将得到的 IP 地址返回给操作系统，同时操作系统也将 IP 地址缓存起来；
  • 操作系统将 IP 地址返回给浏览器，同时浏览器也将 IP 地址缓存起来；

至此，浏览器已经得到了域名对应的 IP 地址。

2.建立连接–三次握手/TCP协议

  • 主机向服务器发送一个建立连接的请求（您好，我想连接）；
  • 服务器接到请求后发送同意连接的信号（好的，你连吧）；
  • 主机接到同意连接的信号后，再次向服务器发送了确认信号（好的，我连了）；

至此，主机和浏览器已经建立连接

3.网页请求与显示/HTTP协议

  • 当服务器与主机建立了连接之后，主机向服务器请求数据，服务器返回相应的数据的过程。
  • 浏览器根据 URL 内容生成 HTTP 请求，请求中包含请求文件的位置、请求文件的方式等等；
  • 服务器接到请求后，会根据 HTTP 请求中的内容来决定如何获取相应的 HTML 文件；
  • 服务器将得到的 HTML 文件发送给浏览器；
  • 在浏览器还没有完全接收 HTML 文件时便开始渲染、显示网页；

在执行 HTML 中代码时，根据需要，浏览器会继续请求图片、CSS、JavsScript等文件，过程同请求 HTML ；

对 DOM 的一些理解

  1. DOM === 想象的树型结构模型.

  2. DOM提供的API的作用就是修改或者查看 HTML 代码

  3. DOM中的O指的是Object,他是在内存中,按照树型结构,通过构造函数(如Node,Element(翻译为标签比较好),Document三个构造函数),构造出对象,将 DOM 展现到内存中

  4. DOM的D指的是Document,可以认为是 HTML 文档

  5. DOM的M指的是Model,因为在 HTML结构 在内存中不好用笔表示,所以用一个模型来表示,这个模型就是树型结构

  6. DOM 树型结构

    1. 上图就是DOM,其中的每个节点(包括矩形节点和椭圆形节点)都是Node类型

    2. document节点是Document构造函数的一个实例对象,document节点代表了整个文档(整个树型结构

      ,我们可以通过直接输入document来获取document节点

    3. html节点是Element构造函数的一个实例对象,html节点又叫根节点

      我们可以通过输入document.documentElement来获得html节点

    4. 椭圆形的文本节点:”你好,我叫饶家俊” 是Text构造函数的一个实例对象(文本节点是Text构造函数的一个实例对象)

数值的进制

  • 十进制：没有前导0的数值。
  • 八进制：有前缀0o或0O的数值，或者有前导0、且只用到0-7的八个阿拉伯数字的数值。

  • 二进制：有前缀0b或0B的数值。

默认情况下，JavaScript 内部会自动将八进制、十六进制、二进制转为十进制

NaN

NaN是 JavaScript 的特殊值，它属于Number类型，但表示“非数字”（Not a Number）

  • 主要出现在将字符串解析成数字出错的场合
  • 另外，一些数学函数的运算结果会出现NaN

String

字符串就是零个或多个排在一起的字符，放在单引号或双引号之中。

转义符

（\）在字符串内有特殊含义，用来表示一些特殊字符，所以又称为转义符,在字符串中不占length

  • \0 ：null（\u0000）
  • \b ：后退键（\u0008）
  • \f ：换页符（\u000C）
  • \n ：换行符（\u000A）
  • \r ：回车键（\u000D）
  • \t ：制表符（\u0009）
  • \v ：垂直制表符（\u000B）
  • \’ ：单引号（\u0027）
  • \” ：双引号（\u0022）

字符串多行书写

在表达字符串时常常因为长度或者易读性问题需要换行，一般连接运算符（+）可以连接多个单行字符串，将长字符串拆成多行书写。输出时为单行

Bool

布尔值代表“真”和“假”两个状态。“真”用关键字true表示，“假”用关键字false表示。布尔值只有这两个值。

6个falsy值

在javaScript中只有以下6个值为false，其他都为true

  • 0
  • ‘’和””
  • false
  • null
  • undefined
  • NaN

Undefined 和 Null

null与undefined都可以表示“没有”，含义非常相似，但在用法上有细微的区别

  • 如果一个变量(非object）没有赋值 -undefined

  • 如果有一个对象（object）没有赋值 -null

Heap堆排序

Posted on 2018-11-01

堆排序的过程

  • 首先让我们创建一个数组:

  • 把数组转换为完全二叉树的结构来表示

  • 比较右下角两个叶节点的数字

  • 选出较大的数字和同支上层父节点数字比较，保证大的在上层，不然则交换

  • 比较最下层相邻的两个叶节点的数字

  • 重复相同的步骤

  • 直到底下两层都满足最大堆

  • 在倒数第二层，比较最右边两个节点的数字，重复相同的过程

  • 被换到倒数第二层的数字，仍要和同支最下层的数字比较，保证大的数字在上层


  • 如此不断重复，直到最大的数字被换到最上层的父节点, 并且下几层都满足最大堆

  • 将最上层父节点的数字和右下角的数字交替位置，并拿出二叉树结构

  • 在数组中最大的数字与最后一个数字交换位置

  • 将二叉树再进行一次最大堆调整，保证最大的数字在最上层

  • 再次将最上层父节点的数字和右下角的数字交替位置，并在数组中交换位置，将最大的数字拿出二叉树结构

  • 如此不断重复上述过程，直到所有数字都被拿出二叉树，数字也排序完成
